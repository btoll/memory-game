# Memory Card Game

Demo: https://www.theowlsnestfarm.com/memory-game/

## Installation

`npm i memory-game`

## Building and Deploying

1. `npm run build`
2. Copy `index.html` and `build/` to your web server.
3. Rub a dog's tummy.
